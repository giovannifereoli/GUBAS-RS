Module integrators 

Source
Expand description

Fixed-step RK4 / ABM, adaptive RK7(8), and LGVI integrators.

Each integrator writes native-endian f64 binary files:

  • output_t/t_out.bin — one f64 per step (time)
  • output_x/x_out.bin — 30 f64s per step (state, row-major)

RK4 and ABM also write perturber positions when the flyby / heliocentric toggles are active (output_h/ and output_sun/).
