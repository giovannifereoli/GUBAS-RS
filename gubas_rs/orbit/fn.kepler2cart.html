Expand description

Convert classical Keplerian elements to Cartesian state [r(3), v(3)] in km and km/s.

Arguments (pointers in C++, plain values here): a = semi-major axis (km), ecc = eccentricity, inc = inclination (rad), raan = right ascension of ascending node (rad), om = argument of periapsis (rad), f0 = true anomaly (rad), g = gravitational parameter G (km³/kg/s²), mu_m = mass of central body (kg).

Returns [rx, ry, rz, vx, vy, vz]. Mirrors vec kepler2cart(...).
