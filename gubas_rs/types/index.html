Shared data structures: Cube, Params, and Initialization.

  • Cube<T> — 3-D array replacing Armadillo’s cube. Flat row-major storage, indexed (l, m, n).
  • Params<T> — all read-only simulation parameters threaded through the ODE and integrators. Generic over Scalar so Dual types can replace f64 for auto-differentiation.
  • promote_params — lift Params<f64> → Params<U> for dual-number AD.
